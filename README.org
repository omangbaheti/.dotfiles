* Commands and Troubleshooting
** Cloning the first time
#+begin_src sh
cd ~
git clone git@github.com:omangbaheti/.dotfiles.git .dotfiles
#+end_src

** Rebuild Configuration
#+begin_src sh 
sudo nixos-rebuild switch --flake .#<hostname> --impure
#+end_src

** Rebuild Homemanager
#+begin_src sh
home-manager switch --flake  ~/.dotfiles/nix-config-org#$<username>@$<hostname>
#+end_src

** Flakes not enabled
- Open /etc/configuration.nix
- add "nix.settings.experimental-features = ["nix-command" "flakes"];"
